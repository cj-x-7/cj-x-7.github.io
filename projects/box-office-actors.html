Box Office Performance by Actor

Interactive starter project showing how an analytics portfolio can move from descriptive views to a strategy recommendation. This demo uses sample data structure only, so you can replace it with a real cleaned dataset later.

Case Study Frame

  • Question: Which actor profiles are driving gross vs ROI?
  • Audience: Studio strategy or content finance teams
  • Output: Filterable dashboard plus recommendation

Project Structure

What this page demonstrates.

The question

Evaluate whether certain actor profiles are associated with stronger top-line performance, more stable returns, or better outcomes in specific budget bands.

The analysis

Compare budget, worldwide gross, ROI, year trend, and actor concentration. Apply filters to simulate how decision-makers would narrow the portfolio.

The strategy take

Translate the pattern into a recommendation on casting premium, franchise dependency, or mid-budget slate design.
